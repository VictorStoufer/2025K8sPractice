1. Copy files into `src/` exactly as named.
2. In your project, import `PlatformerGame.jsx` in App.js and render <PlatformerGame />.
3. `npm start` or `yarn start`.
4. Controls: Arrow keys / A-D to move, Space or Up to jump, Esc to pause.

Notes:
- No external libs required. CSS file included.
- To add sprites, drop images into `src/assets/` and update rendering in Player.render and Enemy.render.
